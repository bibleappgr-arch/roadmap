Offline bible app in greek
==========================

Languages
---------

This document is in english, but issues and pull requests in greek are also accepted.

Bugs
----
Use this repository to make feature requests or submit bugs you found. If you want to submit a bug, [create an issue](https://github.com/bibleappgr/roadmap/issues/new/choose) and give it the bug label.

Bugs get higher priority than feature requests.

Features
--------
If you want to do a feature request, submit a [pull request](https://github.com/bibleappgr/roadmap/pulls) with the feature description to ROADMAP.md. If your pull request attracts enough activity, the feature will be added to the roadmap and eventually the app.
